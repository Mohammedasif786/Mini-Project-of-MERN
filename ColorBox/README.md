A sleek, modern, and high-performance **Random Color Box Generator** built with React and Tailwind CSS. Instantly generate beautiful color palettes with a single click.

---

## ✨ Features

- **🚀 Instant Generation:** Generate multiple random color boxes at once.
- **💎 Hex Code Labels:** Every box displays its unique HEX code.
- **📱 Fully Responsive:** Works perfectly on Mobile, Tablet, and Desktop.
- **🚫 UX Optimized:** - `select-none`: No annoying text highlights while clicking.
  - `draggable="false"`: No accidental ghost-dragging of UI elements.
- **⚡ Fast & Lightweight:** Minimal bundle size with TypeScript for type-safety.

---

## 🛠️ Tech Stack

- **Frontend:** React 18
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** React Context API (optional/implemented)
