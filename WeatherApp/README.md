# 🌦️ Weather App

A sleek and responsive web application that provides real-time weather updates using the OpenWeatherMap API. 

---

## 🚀 Features

* **Real-time Data:** Get live temperature, humidity, and wind speed.
* **Global Search:** Look up weather conditions for any city in the world.
* **Dynamic UI:** The interface changes based on current weather (Sunny, Rainy, etc.).
* **Responsive:** Works perfectly on mobile, tablet, and desktop.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **API:** [OpenWeatherMap API](https://openweathermap.org/)
